# Finance Manager Environment Setup

To run this project locally, you need to set the following environment variables:

## 1. Database
- **DB_PASSWORD**: Your MySQL root password (e.g., `Sathwik@7`)

## 2. Gemini AI
- **GEMINI_API_KEY**: Your Gemini API key (e.g., `AIzaSy...`)

## How to set them (Windows Example):
Run these in your terminal before starting the backend:
$env:DB_PASSWORD="your_password"
$env:GEMINI_API_KEY="your_key"

## How to set them (Linux/Mac Example):
export DB_PASSWORD="your_password"
export GEMINI_API_KEY="your_key"
